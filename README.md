# 🛒 Customer Segmentation using RFM Analysis

## 📌 Project Overview

This project analyzes retail customer transaction data using **RFM (Recency, Frequency, Monetary) Analysis** to identify high-value customer segments.

The objective is to discover the customer groups that contribute the most to revenue and understand their purchasing behavior, providing insights for customer relationship management (CRM) and marketing strategies.

---

## 🎯 Objectives

- Perform customer segmentation using RFM analysis
- Identify high-value customer groups
- Analyze demographic characteristics and purchasing behavior
- Derive business insights for customer retention and marketing

---

## 📂 Dataset

- Retail customer transaction dataset
- Customer demographic information
- Purchase history
- Annual income
- Marital status
- Number of children
- Education level
- Product purchase information
